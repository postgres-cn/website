ABOUT PostgreSQL ACE
PostgreSQL ACE 计划
“PostgreSQL ACE计划”面向以下人群：
PostgreSQL技术大牛，包括但不限于布道者，架构师，DBA、DEV、内核研发等等资深技术人员。
这些人技术娴熟有强烈的意愿分享他们的知识和经验，他们是相应领域的专家，包括但不限于撰写PostgreSQL技术文章、博客或书籍；热心于在各类活动中做PostgreSQL技术分享；编写PostgreSQL插件或扩展功能；发现PostgreSQL bug和提交补丁等。

如何成为PostgreSQL ACE?
希望成为受尊敬的PostgreSQL ACE？如果您一直撰写文章、发表博文、提交代码或提交bug或以其他方式分享PostgreSQL相关知识，即有资格加入。
提名审查
PostgreSQL ACE 提名由 PostgreSQL中国社区核心组成员（根据候选人的专长领域）每个季度审查一次。未能入选的候选人以后还可以再次申请。
